## UEFA Tournaments Dashboard

A high-performance interactive dashboard and web application designed to visualize UEFA tournament results and team locations across Europe. This project features dedicated sections for the UEFA Champions League, UEFA Europa League, and UEFA Conference League, utilizing interactive maps and a modern UI.

---

### Core Features

* **Interactive European Maps:** Integrated Leaflet.js with MarkerCluster to display team locations and stadium information dynamically.
* **Tournament Switcher:** A custom-built floating navigation menu allowing seamless transitions between different tournament boards without returning to the main dashboard.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views. Navigation bars and popups adapt based on screen size to ensure usability on all devices.
* **Performance Focused:** Includes a customized preloader and optimized asset loading to handle large datasets of match results and team metadata.
* **Themed UI:** Each section features unique color schemes and "pulse" animations tailored to the specific tournament's branding (UCL Blue, UEL Orange, UECL Green).

---

### Technologies Used

* **HTML5 & CSS3:** Custom styling with Flexbox, Grid, and advanced CSS animations.
* **JavaScript (ES6+):** Asynchronous data fetching and dynamic DOM manipulation.
* **Leaflet.js:** Open-source JavaScript library for interactive maps.
* **Google Fonts:** Montserrat for a professional and bold typography experience.
